** Task

The /Liturgia Horarum/ (Divine Office) divides the day into canonical
hours of prayer. Your job: write one function per hour, then a single
function that runs through the whole day.

1. Write a separate function for each of these hours:

   | Function | Time     | Label                  |
   |----------+----------+------------------------|
   | lauds    | 6:00 AM  | Morning praise         |
   | terce    | 9:00 AM  | Mid-morning prayer     |
   | sext     | 12:00 PM | Midday prayer          |
   | nones    | 3:00 PM  | Mid-afternoon prayer   |
   | vespers  | 6:00 PM  | Evening prayer         |
   | compline | 9:00 PM  | Night prayer           |

   Each function takes no arguments and prints ONE line in this
   format:

   #+begin_example
    Lauds    - 6:00 AM  - Morning praise
   #+end_example

   Include a *docstring* (one line is enough) describing what each
   function does.

2. Write a function called =pray_day()= that calls all six functions, in
   order, so that running =pray_day()= prints the full schedule.

3. Call =pray_day()= once at the end of your notebook.

4. Challenge: You can also call =pray_day()= in a =main()= function but
   then you need to add this at the end of your script so that Python
   knows that you want to run =main()= directly.
   #+begin_example python
   if __name__ == "__main__":
       main()
   #+end_example
